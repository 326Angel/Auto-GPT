# Kizuki Note

Kizuki Note is a lightweight iPhone note-taking app designed to capture ideas, insights, and small realizations the moment they appear.

The goal is simple: open the app, start writing, and save the thought before it disappears.

## Concept

Most note-taking apps ask users to think about folders, titles, tags, and organization before writing. Kizuki Note takes the opposite approach.

First capture. Organize later.

This project focuses on helping users record raw insights quickly and turn them into useful knowledge, articles, tasks, learning materials, or business ideas in the future.

## Current Scope

The first version is intentionally small.

- iPhone app
- Local storage only
- No login
- No cloud sync
- No AI features in the initial version
- Fast memo input
- Memo list
- Search
- Memo editing

## Why This Matters

Ideas often disappear within seconds. This app is designed for people who frequently create, teach, write, plan, or make decisions.

Target users include:

- Educators
- Learners
- Engineers
- Business professionals
- Consultants
- Creators
- Entrepreneurs

## Technology Plan

- SwiftUI
- SwiftData
- Local device storage
- Face ID support in a future security update

## Security Policy

Kizuki Note treats personal insights as valuable private data.

The MVP stores data only on the device and does not send memo data to external servers. AI features, if added later, will require explicit user approval before any memo is sent to an external API.

Do not commit:

- API keys
- Passwords
- Private certificates
- Real user memo data
- Customer information
- Personal confidential information
