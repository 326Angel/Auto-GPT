# 気づいたときのメモ帳アプリ

## １）プロジェクト概要

「気づいたときのメモ帳アプリ」は、思いついた瞬間、違和感を覚えた瞬間、話したいことが浮かんだ瞬間に、迷わず記録できることを目的とした軽量メモアプリです。

一般的なメモアプリは、書く前に分類、保存場所、タイトル、フォルダを意識させがちです。本プロジェクトでは、最初の目的を「整理」ではなく「捕獲」に置きます。

## ２）コンセプト

### (1) 3秒以内に書き始める

アプリを開いたら、すぐに入力画面が表示されることを重視します。

### (2) 整理はあとでよい

タグ、分類、要約、変換は後工程に回します。

### (3) 気づきを成果物に変える

単なるメモではなく、記事、SNS投稿、音声配信、企画、タスクへ展開できることを目指します。
