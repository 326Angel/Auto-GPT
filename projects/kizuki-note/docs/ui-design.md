# 画面設計

## １）基本思想

アプリを起動した瞬間に書けること。

ホーム画面は作らない。

利用者は考えない。

すぐ書く。

## ２）画面一覧

### Screen01 メモ入力

起動時の画面。

構成：

・大きなテキスト入力欄
・保存ボタン
・一覧ボタン

イメージ

-----------------
気づいたこと

[入力エリア]

保存
一覧
-----------------
