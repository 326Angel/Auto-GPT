# セキュリティ設計

## Phase1

### ローカル保存限定

・端末内のみ保存
・外部通信なし
・クラウド同期なし

### Git管理ルール

保存禁止：

・APIキー
・秘密鍵
・証明書
・実際のメモデータ
・顧客情報
